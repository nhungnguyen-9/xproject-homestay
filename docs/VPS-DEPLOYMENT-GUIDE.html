🚀 Hướng dẫn Triển khai VPS

Nhà Cam Homestay - Production Readiness Guide

Kiến Trúc Hệ Thống

Frontend

Vite + React được build sang static files và phục vụ bởi Nginx.

Backend API

Node.js Express chạy trong Docker container, quản lý bởi PM2 hoặc Docker.

Database

PostgreSQL được triển khai qua Docker Compose với Volume persistent.

Quy Trình 5 Bước Thần Tốc

1

Chuẩn bị VPS (Ubuntu 22.04)

Cập nhật hệ thống và cài đặt Docker + Docker Compose:

sudo apt update && sudo apt upgrade -y
sudo apt install docker.io docker-compose -y
sudo systemctl start docker
sudo systemctl enable docker
2

Cấu hình Biến môi trường (.env)

Tạo tệp .env tại thư mục gốc của VPS:

# Backend
DATABASE_URL=postgres://user:pass@db:5432/nhacam
JWT_SECRET=your_super_secret_key
PORT=3000

# Frontend
VITE_API_URL=https://api.yourdomain.com
VITE_BANK_ACCOUNT=0123456789
3

Khởi chạy với Docker Compose

Chạy lệnh thần thánh để kích hoạt toàn bộ dịch vụ:

docker-compose up -d --build

Hệ thống sẽ tự động build images, thiết lập database và khởi động API.

4

Cấu hình Nginx & SSL

Sử dụng Nginx làm Proxy ngược và cài đặt SSL miễn phí:

sudo apt install nginx certbot python3-certbot-nginx -y
# Cấu hình file /etc/nginx/sites-available/nhacam
sudo certbot --nginx -d yourdomain.com -d api.yourdomain.com
5

Kiểm tra & Vận hành

Kiểm tra logs: docker-compose logs -f
Kiểm tra sức khỏe: curl http://localhost:3000/health
