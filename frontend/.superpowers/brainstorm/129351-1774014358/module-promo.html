Data Model

interface PromoCode {
  id: string;
  code: string; // SUMMER20, VIP50K...
  discountType: 'percent' | 'fixed';
  discountValue: number; // 20 (%) hoặc 50000 (VND)
  maxUses: number;
  usedCount: number;
  startDate: string;
  endDate: string;
  roomTypes: RoomType[] | 'all';
  status: 'active' | 'expired' | 'disabled';
  createdAt: string;
}

// status tự động chuyển 'expired' khi endDate < now hoặc usedCount >= maxUses

Validation (Zod)

Mã: 3-20 ký tự, chỉ A-Z 0-9, unique (không trùng mã đã có)
Giảm %: 1–100
Giảm VND: 1,000 – 10,000,000
Giới hạn lần: 1 – 99,999
Ngày hết hạn: phải sau ngày bắt đầu
Áp dụng booking: Khi user nhập mã → validate mã + kiểm tra loại phòng + kiểm tra còn lượt + kiểm tra hạn
