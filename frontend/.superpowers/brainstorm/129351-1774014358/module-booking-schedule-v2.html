Data Model — Booking mở rộng

interface Booking {
  ...// existing fields
  category: 'guest' | 'internal';
  internalTag?: 'cleaning' | 'maintenance' | 'locked' | 'custom';
  internalNote?: string; // "Sửa điều hòa", "Đang trang trí"
  createdBy?: string; // admin who created
}

Quick Actions — Tạo booking nội bộ

Cách 1: Click ô trống (+) → chọn tab "Nội bộ" trong modal → chọn tag → điền thời gian
Cách 2: Right-click booking khách → "Thêm dọn phòng sau" → auto tạo 30p sau checkout
Dọn phòng nhanh: Khi check-out khách → hỏi "Thêm 30p dọn phòng?" → 1 click tạo
