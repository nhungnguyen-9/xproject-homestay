Data Model

interface Customer {
  id: string;
  name: string;
  phone: string;
  email?: string;
  totalSpent: number; // tính từ booking history
  visitCount: number;
  lastVisit: string; // ISO date
  note?: string; // ghi chú riêng (VIP, sở thích...)
  createdAt: string;
}

// Customer được tự động tạo khi booking mới có SĐT chưa tồn tại
// totalSpent & visitCount tính từ bookingService

Luồng tương tác

Tự động tạo khách: Khi tạo booking mới → nếu SĐT chưa có → tạo Customer mới
Click row: Navigate đến /admin/customers/:id → xem chi tiết + lịch sử
Tìm kiếm: Tìm theo tên, SĐT, hoặc email (debounce 300ms)
Sắp xếp: Gần nhất / Chi tiêu cao nhất / Nhiều lần đặt nhất / A→Z
Sửa ghi chú: Inline edit — click "Sửa ghi chú" → textarea → Lưu
