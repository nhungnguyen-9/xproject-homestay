Cách hoạt động (Frontend-only)

Lưu config: Bot token + chat ID → localStorage (ẩn token khi hiển thị)
Khi tạo booking: bookingService.create() → gọi telegramService.notify() → log vào lịch sử
Mô phỏng: Không gọi API Telegram thật — chỉ render preview + log "Đã gửi" vào bảng lịch sử
Gửi test: Nút "Gửi tin test" → hiện toast + thêm entry vào log
Khi có backend: telegramService.notify() → gọi API server → server gọi Telegram Bot API thật
